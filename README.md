**1_ResidenceTime.ipynb:** Illustrates how to find the residence time of a water in the hydration shell of other water

**2_JumpReorientationTime_Absorbing.ipynb:** For calculating the jump reorientation time, using absorbing condition (using [[2]](<https://www.science.org/doi/10.1126/science.1122154>))

**2_FrameReorientation_Absorbing.ipynb:** For finding the frame reorientation time, using absorbing conditions, of O-O (using [[2]](<(https://www.science.org/doi/10.1126/science.1122154)>))

**2_ReorientationTime.ipynb:** For calculating the reorientation time of water molecules (using [[2]](<https://www.science.org/doi/10.1126/science.1122154>))
